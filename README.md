Marketing Campaign Analytics 

An end-to-end data analytics project that analyzes marketing campaign data to uncover performance trends, customer behavior, ROI efficiency, and growth opportunities using SQL and Power BI.

🚀 Project Overview

This project simulates a real-world marketing analytics scenario where raw campaign data is transformed into actionable insights.

The focus is not just on building dashboards, but on answering key business questions related to campaign performance, targeting, and profitability.

🎯 Business Questions Solved

What is the overall performance of marketing campaigns?

Which channels generate the highest ROI?

Which audience segments respond the best?

How effective are different campaign types?

Is marketing performance improving over time?

Which locations offer the best opportunities?

Are campaigns cost-efficient?

Do a few campaigns drive most of the results?

🧰 Tools & Skills

PostgreSQL → Data cleaning, transformation, advanced analysis

Power BI → Dashboard design, storytelling, visualization

DAX → KPI calculations (CTR, ROI, Conversion Rate, Cost Efficiency)

🔍 Key Insights

1.Marketing campaigns are highly profitable with an average ROI of ~5

2.CTR (~10%) and conversion rate (~7%) indicate strong funnel efficiency

3.Performance across channels and segments is highly consistent

4.Facebook and Website channels slightly outperform others

5.Influencer campaigns show marginally better returns

6.Men aged 25–34 generate the highest ROI

7.ROI fluctuates over time, indicating seasonal impact

8.A small group of campaigns contributes higher returns (Pareto principle)

9.Geographic performance shows minimal variation across locations

10.Overall strategy is stable but lacks differentiation

🚀 Recommendations

1.Focus budget on high-performing channels (Facebook, Website)

2.Improve underperforming channels like Instagram

3.Target high-value audience segments (Men 25–34)

4.Implement A/B testing to improve campaign differentiation

5.Optimize campaigns based on seasonal performance trends

6.Scale high-performing campaigns for better returns

7.Introduce advanced segmentation for personalized marketing

8.Monitor cost efficiency and reduce unnecessary spend

9.Shift from broad targeting to data-driven strategies
