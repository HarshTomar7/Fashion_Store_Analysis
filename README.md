📊 Marketing Campaign Analytics Dashboard | SQL + Power BI

An end-to-end data analytics project that analyzes large-scale marketing campaign data to uncover performance trends, ROI efficiency, and customer behavior for data-driven decision-making.

🚀 Project Overview

This project simulates a real-world marketing analytics scenario where campaign data is transformed into actionable insights.

The focus is not just on visualization, but on solving key business problems related to performance optimization, targeting, and profitability.

🎯 Problem Statement

Businesses invest heavily in marketing campaigns but often lack clarity on which strategies drive the best results.

This project aims to analyze campaign performance, identify high-value segments, and optimize marketing spend using data-driven insights.

🎯 Business Questions Solved

What is the overall performance of marketing campaigns?

Which channels generate the highest ROI?

Which audience segments respond the best?

How effective are different campaign types?

Is performance improving over time?

Which campaigns drive the most value?

Where can marketing strategy be optimized?

🧰 Tools & Skills

PostgreSQL → Data cleaning, transformation, analysis

Power BI → Dashboard design & storytelling

DAX → KPI calculations (CTR, ROI, Conversion Rate, Cost Efficiency)

📁 Dataset

200,000+ campaign records

110M+ clicks and 1.1B+ impressions

Includes metrics like ROI, conversion rate, cost, and engagement

Covers multiple dimensions: channel, audience, campaign type, and location

Key Insights & Recommendations

👉 Focuses on decision-making and optimization strategies

🔍 Key Insights

1.Campaigns are highly profitable with an average ROI of ~5

2.Strong funnel performance with ~10% CTR and ~7% conversion rate

3.Minimal variation across channels and segments indicates standardized strategy

4.Facebook and Website channels slightly outperform others

5.Men aged 25–34 generate the highest ROI

6.ROI fluctuates over time, indicating seasonal impact

7.A small group of campaigns drives a large share of returns (Pareto principle)

8.Geographic performance shows limited variation

🚀 Recommendations

1.Increase investment in high-performing channels (Facebook, Website)

2.Optimize underperforming channels like Instagram

3.Focus on high-value audience segments (Men 25–34)

4.Implement A/B testing to improve campaign differentiation

5.Optimize campaigns based on seasonal trends

6.Scale high-performing campaigns for maximum ROI

7.Shift towards data-driven, targeted marketing strategies

📊 Key Metrics

CTR = Clicks / Impressions

Conversion Rate = Conversions / Clicks

ROI = Return on Investment

Cost per Click = Cost / Clicks
